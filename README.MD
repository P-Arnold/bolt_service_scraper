# Bolt Scraper

## Setup

Requires Python 3.9+

1. Clone repository
2. Navigate to directory
3. Create virtual environment using venv, and activate
   
    `python -m venv venv`

    `source venv/bin/activate`
4. Install requirements
   
    `python -m pip install -r requirements.txt`

5. Run script

    `python src/main.py`
